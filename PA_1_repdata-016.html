This is an R Markdown document written for the Coursera course on Reproducible Research, repdata-16, Dec 2014. The data set is from a collection of personal activity data,activity.csv, recorded by personal fitness devices and may be found at:

            https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2Factivity.zip

The data consists of two months of data from an anonymous individual collected during the months of October and November, 2012 and include the number of steps taken in 5 minute intervals each day. To run the script the data should be downloaded to the working directory. The variables included in this dataset are:

steps: Number of steps taking in a 5-minute interval (missing values are coded as NA)

date: The date on which the measurement was taken in YYYY-MM-DD format

interval: Identifier for the 5-minute interval in which measurement was taken

The dataset is stored in a comma-separated-value (CSV) file and there are a total of 17,568 observations in this dataset. The first five rows are:

steps date interval 1 NA 2012-10-01 0 2 NA 2012-10-01 5 3 NA 2012-10-01 10 4 NA 2012-10-01 15 5 NA 2012-10-01 20 6 NA 2012-10-01 25
